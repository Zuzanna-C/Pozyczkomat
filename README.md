# Pożyczkomat

**Pożyczkomat** is a desktop application developed for thw **Delphi Hackathon 2025**, organized by **BSC Polska**. It allows users to track and manage their financial allowances in a simple and intuitive way.

## Requirements

To run the application, ensure the following:

- Include the required **SQLite DLL files** (e.g., `sqlite3.dll`) in the application's root directory.
